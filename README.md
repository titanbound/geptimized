# Geptimized

**Geptimized** is a cross-platform, performance-optimized Electron wrapper for NVIDIA GeForce NOW. Built for low latency, high responsiveness, and extended stability during long cloud gaming sessions. Ideal for users who demand minimal frame drops, low input lag, and network optimizations ‚Äî especially on systems using GeForce NOW.
